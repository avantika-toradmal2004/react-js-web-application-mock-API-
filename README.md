# react-js-web-application-mock-API

A modern React.js web application built using mock API data.  
This project demonstrates React fundamentals, API integration, component-based architecture, and clean UI design.

---

##  Features
- React functional components
- API integration using Mock API
- Dynamic data rendering
- Responsive UI
- Clean and modular code structure
- Easy to understand project flow

---

##  Tech Stack
- **Frontend:** React JS
- **Styling:** CSS / Bootstrap
- **API:** Mock API
- **Version Control:** Git & GitHub
